# Мастер-CRM

## What This Is

Мультимессенджер CRM для самозанятых мастеров (beauty, репетиторы, фитнес, ремонт и др.), работающая одновременно в Telegram, MAX и VK. Клиенты записываются через мини-апп в мессенджере, мастер управляет расписанием через мини-апп или веб-панель. Главная фишка — после оплаты через СБП автоматически формируется чек в "Мой Налог" через Robokassa Robochecks.

## Core Value

Мастер нажимает "Завершить" — клиент получает ссылку на оплату СБП — чек в "Мой Налог" формируется автоматически. Вместо 9 ручных шагов — 3.

## Constraints

- **Стек бэкенд**: Python FastAPI + PostgreSQL — выбрано из опыта с другими проектами
- **Стек фронтенд**: React + TypeScript — единый мини-апп для TG/MAX/VK + веб-панель
- **Боты**: aiogram (TG), MAX Bot API (нативный), VK Bot API
- **Платежи**: Robokassa — уже зарегистрирована, СБП + Robochecks
- **Хостинг**: VPS (свой сервер) + Docker Compose
- **Идентификация**: Номер телефона как primary key клиента
- **Юрлицо**: ИП (подходит для MAX и Robokassa)
