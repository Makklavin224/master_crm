  3. **backend/app/services/portfolio_service.py**: Create new file with class `PortfolioService`:

     ```python
     import uuid
     from pathlib import Path
     from PIL import Image
     import io

     ALLOWED_CONTENT_TYPES = {"image/jpeg", "image/png", "image/webp"}
     EXTENSION_MAP = {"image/jpeg": ".jpg", "image/png": ".png", "image/webp": ".webp"}

     class PortfolioService:
         @staticmethod
         async def process_upload(
             master_id: uuid.UUID,
             file_content: bytes,
             content_type: str,
             settings,  # app.core.config.Settings
         ) -> tuple[str, str]:
             """Process upload: validate, resize, create thumbnail, save to disk.

             Returns (file_path, thumbnail_path) relative paths for DB storage.
             Raises ValueError on validation failure.
             """
     ```

     Implementation details:
     - Validate `content_type` in `ALLOWED_CONTENT_TYPES`, raise ValueError("Формат файла не поддерживается. Допустимые: JPEG, PNG, WebP") if not.
     - Validate `len(file_content) <= settings.portfolio_max_file_size`, raise ValueError("Файл слишком большой. Максимум 5 МБ") if exceeded.
     - Open image with `Image.open(io.BytesIO(file_content))`. Convert RGBA to RGB if mode is RGBA (for JPEG compat). Use `.verify()` first on a copy to detect corrupt files, then re-open for processing.
     - Generate unique filename: `{uuid4}{ext}` where ext from EXTENSION_MAP.
     - Master directory: `{settings.portfolio_base_path}/{master_id}/`
     - Create directory if not exists: `Path(master_dir).mkdir(parents=True, exist_ok=True)`
     - **Full-size**: `image.thumbnail((settings.portfolio_full_size, settings.portfolio_full_size), Image.LANCZOS)` -- modifies in place, preserves aspect ratio, only downsizes. Save with quality=85 for JPEG, optimize=True.
     - **Thumbnail**: Create a copy, `thumb.thumbnail((settings.portfolio_thumb_size, settings.portfolio_thumb_size), Image.LANCZOS)`. Save as `thumb_{filename}` in same directory. Same quality settings.
     - Return relative paths: `("{master_id}/{filename}", "{master_id}/thumb_{filename}")`
     - Use `asyncio.to_thread()` to wrap the synchronous Pillow/IO operations.

     Add static method:
     ```python
     @staticmethod
     async def delete_files(file_path: str, thumbnail_path: str, settings) -> None:
         """Delete photo files from disk. Silently ignores missing files."""
     ```
     Uses `Path(settings.portfolio_base_path / file_path).unlink(missing_ok=True)` for both paths. Also wrapped in `asyncio.to_thread()`.
