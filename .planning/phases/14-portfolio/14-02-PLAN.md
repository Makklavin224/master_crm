  3. **public/src/components/PortfolioSection.tsx**: Create new component.

     Props: `{ photos: PortfolioPhoto[] }`.

     If `photos.length === 0`, return `null` (hide section entirely).

     **State:**
     - `selectedTag: string | null` (null = show all)
     - `lightboxIndex: number | null` (null = closed)

     **Derive:**
     - `tags`: unique non-null service_tags from photos: `[...new Set(photos.map(p => p.service_tag).filter(Boolean))]`
     - `filteredPhotos`: if selectedTag, filter by it; otherwise all photos

     **Layout (Tailwind classes matching existing public page style):**

     ```
     <section className="px-4 py-6">
       <h2 className="text-lg font-semibold text-text-primary mb-4">–ü–æ—Ä—Ç—Ñ–æ–ª–∏–æ</h2>

       {/* Tag filter pills */}
       {tags.length > 0 && (
         <div className="flex gap-2 mb-4 overflow-x-auto scrollbar-hide">
           <button onClick={() => setSelectedTag(null)}
             className={pill classes, active when selectedTag === null}>
             –í—Å–µ
           </button>
           {tags.map(tag => (
             <button key={tag} onClick={() => setSelectedTag(tag)}
               className={pill classes, active when selectedTag === tag}>
               {tag}
             </button>
           ))}
         </div>
       )}

       {/* Horizontal scroll gallery */}
       <div className="flex gap-3 overflow-x-auto scrollbar-hide pb-2">
         {filteredPhotos.map((photo, index) => (
           <button key={photo.id} onClick={() => setLightboxIndex(index)}
             className="flex-shrink-0 rounded-xl overflow-hidden">
             <img
               src={`/api/v1/media/${photo.thumbnail_path}`}
               alt={photo.caption || "–Ý–∞–±–æ—Ç–∞ –º–∞—Å—Ç–µ—Ä–∞"}
               className="w-28 h-28 object-cover"
               loading="lazy"
             />
           </button>
         ))}
       </div>

       {/* Lightbox overlay */}
       {lightboxIndex !== null && (
         <Lightbox>...</Lightbox>
       )}
     </section>
     ```

     **Tag pill styles:**
     - Inactive: `px-3 py-1.5 rounded-full text-sm bg-surface text-text-secondary border border-border whitespace-nowrap`
     - Active: `px-3 py-1.5 rounded-full text-sm bg-accent text-white whitespace-nowrap`
     - Match the PillButton/filter pill patterns from the existing codebase.

     **Lightbox implementation** (inline, no external library):
     - Fixed overlay: `fixed inset-0 z-50 bg-black/90 flex items-center justify-center`
     - Close button (X) top-right: `absolute top-4 right-4 text-white text-2xl`. Use Lucide `X` icon.
     - Image: `<img src={/api/v1/media/${filteredPhotos[lightboxIndex].file_path}} className="max-w-full max-h-[85vh] object-contain" />`
     - Left/Right navigation: Lucide `ChevronLeft` and `ChevronRight` icons, positioned absolute left-4 / right-4, centered vertically. Only show if there are more photos in that direction.
     - Click left arrow: `setLightboxIndex(prev => Math.max(0, prev - 1))`
     - Click right arrow: `setLightboxIndex(prev => Math.min(filteredPhotos.length - 1, prev + 1))`
     - Click overlay background: close lightbox.
     - Keyboard: `useEffect` with `keydown` listener: Escape -> close, ArrowLeft -> prev, ArrowRight -> next.
     - Caption below image if exists: `<p className="text-white text-center mt-2">{photo.caption}</p>`.
     - Prevent body scroll when lightbox is open: `useEffect` that sets `document.body.style.overflow = 'hidden'` and restores on cleanup.
     - Touch/swipe support for mobile: track `onTouchStart` and `onTouchEnd` x-coordinates. If swipe distance > 50px, navigate prev/next.
