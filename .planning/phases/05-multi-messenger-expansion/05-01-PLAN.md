       ```python
       def validate_max_init_data(

       ```

       `validate_vk_launch_params(query_string, app_secret) -> dict | None`:
       Implement per VK official algorithm:
       - Parse all query params from query_string
       - Extract "sign" param
       - Filter params starting with "vk_"
       - Sort alphabetically by key
       - URL-encode sorted params (using OrderedDict for determinism)
       - HMAC-SHA256(app_secret.encode(), encoded_string.encode())
       - base64-encode result, strip trailing "=", replace "+" with "-", "/" with "_"
       - Compare with received sign using hmac.compare_digest (timing-safe)
       - Return flattened dict of all params on success, None on failure
       Import: `import base64` and `from collections import OrderedDict` (add at top).
