      2. Calculate ends_at = starts_at + service.duration_minutes.
      3. SELECT FOR UPDATE: check for overlapping confirmed/pending bookings. If found, raise HTTPException 409 "This time slot is no longer available".
      4. Call find_or_create_client(db, client_name, client_phone, source_platform, tg_user_id).
      5. Call get_or_create_master_client(db, master_id, client.id).
      6. Create Booking row (status="confirmed").
      7. Call update_visit_stats().
      8. flush() to trigger exclusion constraint check.
      9. Return booking with loaded relationships.
    - `cancel_booking(db, booking_id, cancelled_by, master_id=None, cancellation_deadline_hours=None) -> Booking`:

      - SELECT FOR UPDATE overlap check for new time.
      - Update starts_at, ends_at.
      - Return updated booking.
    - `create_manual_booking(db, master_id, service_id, starts_at, client_name, client_phone, notes=None) -> Booking`:
