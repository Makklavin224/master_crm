API endpoints:
- GET /api/v1/masters/{identifier}/profile -> MasterProfile
- GET /api/v1/masters/{identifier}/services -> ServiceRead[]
- GET /api/v1/masters/{identifier}/slots?date=YYYY-MM-DD&service_id=UUID -> AvailableSlotsResponse
- GET /api/v1/masters/{identifier}/reviews -> ReviewRead[]

Design spec (Section 1):
- Hero: avatar 96px circle, name, specialization, city, rating stars + count
- Services: cards with name, price (rub), duration (min), "Zapisatsya" button each
- Nearest slots: 3-5 days ahead, first free slots
- Reviews: last 10, average rating at top, individual entries
- Contacts: messenger links, address
- Sticky footer: fixed "Zapisatsya" button visible on scroll
